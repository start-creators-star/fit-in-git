# fit-in-git



## Aufgabe

1) Klone das Projekt über https.
2) Erstelle von deinem Gruppen-Branch (gruppe_eins/ gruppe_zwei) deinen persönlichen Branch (vorname_nachname)
3) Füge eine Änderung in deiner zugeordneten Textdatei (Blau, Gelb, Grün, Rot) hinzu.
4) Add, commit (mit Message) und push deine Änderungen (commits).
